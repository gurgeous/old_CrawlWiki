Conjurers start with a Book of Conjurations.

Wizards start with a Book of Minor Magic.

Warpers start with a Book of Spatial Translocations.

Venom Mages start with a Young Poisoner's Handbook.

Transmuters start with a Book of Changes.

Necromancers start with a Book of Necromancy.

Summoners start with a Book of Callings.

Enchanters start with a Book of Maledictions.

Fire Elementalists start with a Book of Flames.

Ice Elementalists start with a Book of Frost.

Air Elementalists start with a Book of Air.

Earth Elementalists start with a Book of Geomancy.

Skalds start with a Book of Battle.

Wanderers may start with a random starting-level spell book.

All other classes begin without a spell book.
