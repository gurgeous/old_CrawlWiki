A deck of magical cards which induce changes in the user or his environment.

The deck of changes selects its cards from the archetype decks across versions according to the following table, selecting each deck with equal probability. Since archetype decks contain varying numbers of cards, the probabilities for individual cards in the physical deck are not necessarily equal.
