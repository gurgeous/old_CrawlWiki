Casting noise 3

Enslaves any undead in the vicinity of the caster.

Control Undead is a level 4 Necromancy spell, which causes all nearby undead to be affected as if by enslavement. Being a mass enchantment, it has a +50% bonus to the final spellpower, which makes this spell much more effective than enslavement. The effect is temporary, and can be shrugged off by magic resistance.

History

  • Control Undead was removed in 0.21.
  • Prior to 0.10, Control Undead was a level 5 Hexes-Necromancy spell.
