These pages have new artwork being added in 0.15. When the new version arrives, do the following:

  • Save a copy of the old artwork with "Old_-_" added before the file name.
  • Upload the newly named file using the "Upload file" button in the Toolbox on the left.
  • Update the Retired Artwork page with the new entry.
  • Grab the updated artwork from the tiles directory. You'll want to save it under the name of the original artwork.
  • Upload the new artwork to the site using the "Upload file" button in the toolbox on the left.
  • Check to make sure that the new artwork is loading on its page properly.
