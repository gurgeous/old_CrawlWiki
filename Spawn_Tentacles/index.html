Spawn Tentacles

From CrawlWiki
Jump to: navigation, search
Version 0.16: This article may not be up to date for the latest stable release of Crawl.

Spawn Tentacles is a monsters-only spell which "creates" a number of tentacles which will chase you down, even on land. These tentacles share an HP pool with the caster itself, however, so attacking them is not a wasted effort.

The following monsters can cast Spawn Tentacles:

  • Kraken
  • Tentacled starspawn
